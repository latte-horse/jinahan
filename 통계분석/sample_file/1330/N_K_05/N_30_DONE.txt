루피움, 추석 프로모션 오픈 "다음달 2일부터 순차적 발송"
루피움, 추석 프로모션 오픈 /루피움 공식 SNS
프리미엄 코스메틱 브랜드 루피움이 추석 프로모션을 오픈한다.
루피움은 28일 단 하루 추석 선물세트와 단품을 30% 할인하는 '추석 프로모션'을 실시한다.
오전 11시에 오픈되며, 이날 주문건은 다음달 2일부터 순차적으로 발송될 예정이다.
'추석선물세트1'은 리얼 모이스처 퍼퓸 바디 크림 1개 + 클렌저 2개 + 쇼핑백 세트로 구성됐으며, '추석선물세트2'는 리얼 모이스처 퍼퓸 바디 크림 1개 + 클렌저 1개 + 루피움 데일리 소프트 인티메이트 워시 1개 + 쇼핑백 세트로 구성됐다.
'추석선물세트3'은 리얼 모이스처 퍼퓸 바디 클렌저1개 + 패밀리 샤워 필터 (그린티, 밀크라이스) 1+1 + 쇼핑백 세트, '추석선물세트4'는 리바이빙 앰플 1개 + 루피움 리바이빙 에센셜 세럼 1개 + 쇼핑백 세트로 구성됐다.
라이스 딥 클렌저 크림 마스크 1개 + 루피움 비타민 클렌징 밤 100ml 1개 + 쇼핑백으로 구성된 '추석선물세트5' 역시 눈길을 모은다.
한편 자세한 사항은 루피움 공식 홈페이지 및 인스타그램에서 확인 가능하다.
